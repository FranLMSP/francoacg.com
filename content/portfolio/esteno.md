**RMG-001** is a Gameboy emulator written in Rust. It has the basic Gameboy emulator functionalities
such as the CPU, the PPU, some cartridges MBCs (Memory Bank Controllers), save files, and the Gameboy
timer.

#### Tools used:
* Codeigniter
* MariaDB
* jQuery
* Bootstrap
* DataTables
